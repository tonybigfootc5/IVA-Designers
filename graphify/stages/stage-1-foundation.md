# Stage 1 — Foundation & Scaffolding

## Goal

Establish the web platform shell, public SEO surfaces, PWA support, and the deployment baseline.

## Deliverables

- Next.js 15 App Router application
- Tailwind design system
- PWA manifest and service worker
- Public landing, course, pricing, and blog pages
- Prisma schema baseline
- GitHub CI workflow
- Vercel deployment config

## Exit Criteria

- `npm run lint` passes
- `npm run build` passes
- Vercel can build from the repository without manual command changes
