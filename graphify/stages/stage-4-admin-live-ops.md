# Stage 4 — Admin & Live Operations

## Goal

Give operators enough control to manage catalog, offers, and live sessions.

## Deliverables

- Admin overview dashboard
- Course, module, lesson, and coupon creation flows
- Live session scheduling
- Admin operational APIs marked dynamic for Vercel-safe execution

## Exit Criteria

- Admin UI can create content structures and offers
- Admin API routes build safely in CI and Vercel
